# Default Baseline Data

This dataset contains age-specific cancer penetrance rates for both
females and males. As an example, the data is derived from the SEER
program for colorectal cancer females and males.

## Usage

``` r
baseline_data_default

baseline_data_default
```

## Format

A data frame with 94 rows and 3 variables:

- Age:

  Age in years

- Female:

  Female baseline risk

- Male:

  Male baseline risk

A data frame with 94 rows and 3 variables:

- Age:

  Age in years (1 to 94)

- Female:

  Penetrance rate for females

- Male:

  Penetrance rate for males
