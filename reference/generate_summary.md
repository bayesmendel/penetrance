# Generate Summary

Function to generate summary statistics

## Usage

``` r
generate_summary(data, verbose = FALSE)
```

## Arguments

- data:

  A list with combined results.

- verbose:

  Logical, whether to print summary to console. Default is FALSE.

## Value

A data.frame containing summary statistics (min, 1st quartile, median,
mean, 3rd quartile, max) for each parameter.
